These UI components are intentional design-time forks of the shadcn/ui primitives also
used in `artifacts/shopify-agent/src/components/ui/`. The mockup-sandbox has additional
components (sidebar, chart, calendar, etc.) and minor variant tweaks (button, badge) that
support design mockup workflows. These copies should NOT be auto-synced with shopify-agent;
changes in one artifact do not automatically apply to the other.
